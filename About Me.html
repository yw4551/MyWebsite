About Me

Hi, my name is Yitzi Weiner, and I’m a junior QA engineer. I am familiar with both types of testing: manual testing and automation testing. I can run tests on all types of platforms and services, like web and mobile testing. I can run tests on the client side and on the server side.
The reason I chose the QA position is because I always had a strong pull toward anything that had to do with high tech and technology, and since I remember myself, I learned how to code in different languages on my own, and then I realized that I'm very good at tracing the small details, so because there are two topics that are close to my heart, I chose this career.
I have studied a high-level course of QA in the College of Automation by Gal Matalon (www.automation.co.il/). The course was split into two parts. In the first part, we studied manual testing, and we learned how to prepare for the testing by writing the STP and STD documents, writing the test cases, and how to perform the tests and write the STR document.
In the second part of the course, we studied automation testing, starting with learning the Java programming language, and then we learned how to write code with Selenium and use the most recent tools to run automation tests, like Maven to organize the plugins and to run the tests from the command line, Jenkins to run a test in advance, environment testing with Docker, and all of the new and most recent versions of tools that you can use for automation testing.
